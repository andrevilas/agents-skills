# Scope Baseline Publisher

Skill canônica para leitura de working scope, change sets e publicação de baseline sobre o MCP `lpm`.

## Objetivo

Transformar o agente em um operador capaz de:

- revisar drift entre baseline vigente e working scope
- preparar e atualizar change sets com contexto de materialidade
- publicar baselines com guardrails explícitos
- preservar a governança de escopo entre requisitos e execução

## Capacidades Centrais

- working scope review
- change set drafting
- publication gate handling
- baseline publication

## Princípios

- tratar baseline vigente como referência executiva
- não publicar escopo sem ler o delta atual
- usar materialidade e review formal como guardrails reais
- manter claro o que é rascunho, aprovado e publicado

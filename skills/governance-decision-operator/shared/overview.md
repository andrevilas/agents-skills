# Governance Decision Operator

Skill canônica para operar governance threads, decision inbox e outcomes estruturados sobre o MCP `lpm`.

## Objetivo

Transformar o agente em um operador capaz de:

- revisar threads de governança com contexto completo
- responder ou resolver decisões com outcomes auditáveis
- derivar activity, requirement ou change set quando a decisão exigir
- manter lineage explícita entre pergunta, decisão e ação operacional

## Capacidades Centrais

- decision inbox triage
- governance thread resolution
- structured outcome mapping
- auditable follow-up

## Princípios

- não responder decisão sem ler contexto e histórico da thread
- preservar outcome estruturado, não só texto livre
- escolher o menor desdobramento operacional compatível com a decisão
- manter governança separada de execução cotidiana

# Prompt Recipes

Use these examples when the user wants the skill invoked in a focused operating mode.

## Kickoff

`Use $software-project-orchestrator in kickoff mode to create and structure a new project in LPM. Set up the project, team, milestones, and label taxonomy with minimal duplication.`

## Planning

`Use $software-project-orchestrator in planning mode to prepare the next cycle. Review backlog, assign owners, set cycle membership, and map explicit dependencies.`

## Execution

`Use $software-project-orchestrator in execution mode to review active work, identify stalled issues, comment where updates are needed, and escalate deadline risk.`

## Monitoring

`Use $software-project-orchestrator in monitoring mode to analyze project health with LPM analytics. Report completion rate, average completion time, burndown trend, and main delivery risks.`

## Optimization

`Use $software-project-orchestrator in optimization mode to identify systemic blockers, assignment imbalance, and workflow friction, then recommend or execute the highest-leverage adjustments.`

## Portfolio Hygiene

`Use $software-project-orchestrator to review whether this project should remain active, be archived, or be restructured, and justify the decision from current project state and analytics.`

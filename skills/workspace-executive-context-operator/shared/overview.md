# Workspace Executive Context Operator

Skill canônica para operar o contexto executivo do workspace sobre o MCP `lpm`.

## Objetivo

Transformar o agente em um operador capaz de:

- revisar contexto executivo, gaps e readiness do workspace
- validar alinhamento entre cobertura do workspace e projetos
- propor operating model e routing policy com estrutura auditável
- aplicar mudanças controladas no contexto executivo do workspace

## Capacidades Centrais

- executive context review
- readiness validation
- workspace/project alignment
- controlled context updates

## Princípios

- ler o contexto executivo antes de propor qualquer mutação
- tratar gaps de governança como sinais estruturais, não ruído
- separar draft, validação e aplicação de mudanças sensíveis
- preservar auditabilidade de toda alteração executiva
